常见问题

App 收集我的位置数据吗？

只在你主动开启路线规划、导航、组队骑行时使用精确位置。位置数据保存在你的 iCloud 私有数据库（开发者无法访问），不上传任何第三方服务器，不用于广告追踪。 详见隐私政策。

组队骑行如何工作？

近距离（<100m）通过 Apple MultipeerConnectivity 蓝牙/WiFi 直连，免流量； 远距离通过 iCloud 加密同步。生成 6 位邀请码，队友输入即可加入，1 个房间最多 10 人。

没有网络能用吗？

离线优先设计。除路线规划、天气、组队远距离同步等需要联网的功能外， 胎压管理、油耗记录、车机模式 HUD、车库管理等核心功能均可离线使用。

支持哪些摩托车型？

内置 23 款主流型号车型库（本田 / 雅马哈 / 川崎 / 杜卡迪 / 哈雷 / BMW / KTM / 凯旋等）， 也可手动新建任意车型并自定义外形、颜色、胎压目标。

会员订阅可以退订吗？

可以。任何时候都可在 iOS 系统设置 → Apple ID → 订阅 中取消。 取消后剩余订阅期内仍可使用高级功能。
